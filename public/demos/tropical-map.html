Guide de la carte

touch_app
Naviguer sur la carte

Faites glisser la carte pour vous déplacer. Utilisez la molette ou pincez pour zoomer et dézoomer. Les boutons + et − à droite permettent aussi de zoomer.

place
Cliquer sur un pin

Cliquez sur un repère (vert = résidence, rouge = commerce) pour afficher ses informations. Le panneau latéral s'ouvre avec le détail du lieu et son code Tropical Map.

search
Rechercher un lieu

Tapez un nom, un code TM ou une adresse dans la barre de recherche, puis validez. La carte se centre sur le résultat et affiche les lieux correspondants.

filter_list
Filtrer les lieux

Ouvrez le panneau latéral (menu ☰) pour filtrer par type, commune ou quartier. Les catégories sous la recherche permettent de filtrer rapidement.

add_location
Ajouter ou signaler

Cliquez sur le bouton vert add_location pour ajouter un lieu, signaler un incident ou publier une annonce. Les chips (Incident, Accident, Travaux…) permettent de placer un signalement directement.

layers
Vue carte ou satellite

Utilisez le bouton couches pour basculer entre carte routière et vue satellite. L'icône soleil permet de passer en mode jour ou nuit.

my_location
Ma position

Cliquez sur l'icône boussole pour centrer la carte sur votre position GPS (autorisez la géolocalisation si demandé).
